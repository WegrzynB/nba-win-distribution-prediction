# The use of regression models to predict NBA teams’ win distributions

## ENG
The aim of this paper is to present the process of building various regression models using the R programming language.
The study will employ linear regression, multiple regression, and Poisson regression models. Based on statistical data from previous NBA seasons made available by the league, these models will be used to estimate the number of wins achieved by individual teams in subsequent seasons. The leaps and MASS packages will be applied, along with tools that enable the automation of variable selection. In addition, Poisson regression models will be developed using the aforementioned packages.
The paper will also demonstrate how statistical methods can be used to determine which of the developed models achieves the highest predictive accuracy.

## PL
Celem pracy będzie przedstawienie procesu budowy różnych modeli regresji przy użyciu języka R. W pracy wykorzystamy modele regresji liniowej, regresji wielorakiej oraz regresji Poissona. Bazując na udostępnianych przez ligę NBA danych statystycznych z wcześniejszych sezonów, wykorzystamy te modele do próby oszacowania liczby zwycięstw poszczególnych zespołów ligi w kolejnych sezonach. Zastosujemy biblioteki leaps oraz MASS oraz narzędzia, które umożliwią automatyzację wyboru zmiennych. Dodatkowo, zbudujemy modele regresji Poissona, również z wykorzystaniem powyższych bibliotek. Pokażemy też jak za pomocą metod statystycznych ustalić, który ze zbudowanych modeli osiąga największą dokładność.
